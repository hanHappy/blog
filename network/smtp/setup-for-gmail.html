Gmail에서 SMTP를 위한 설정하기 ​

Gmail SMTP를 활용하여 회원가입, 비밀번호 찾기 등에 필요한 이메일 발송 기능을 구현해보겠습니다.

(배경: 웹 호스팅 메일 사용하는 경우 비용이 발생. 후이즈 참고)

SMTP란?

Simple Mail Transfer Protocol 인터넷에서 이메일을 보내기 위해 이용되는 프로토콜로, TCP 포트 번호 25번을 사용합니다.


텍스트로 보기가 어려우신 분들은 Youtube에 how to setup gmail SMTP로 검색해보시면 자세하게 설명된 영상이 많습니다.

1. Google 계정 2단계 인증 설정하기 ​

계정 관리 페이지로 이동합니다.

보안 탭에서 2단계 인증을 설정합니다.

2. 앱 비밀번호 설정하기 ​

설정을 완료하면 위와 같이 결과를 확인할 수 있습니다. 해당 링크를 다시 클릭하여 앱 비밀번호를 설정하겠습니다.

앱 비밀번호 클릭

원하는 앱 이름을 설정합니다. 저는 [프로젝트명]-SMTP로 설정하였습니다.

만들기를 클릭하면 앱 비밀번호가 표시된 모달이 나타납니다. 메일을 전송할 때 필요한데요, 안내 문구에 따르면 비밀번호는 기억하지 않아도 되므로 적어두지 않겠습니다.

짜잔~ 앱 비밀번호가 생성되었습니다. 그러면 잘 동작하는지 확인해보겠습니다.

3. 테스트하기 ​

“SMTP test tool”을 검색하여 해당 사이트에 접속합니다.

필요한 정보를 입력한 뒤 Test it!

  • SMTP Server: smtp.gmail.com
  • Port: 465 (또는 587)
  • Security: SSL
  • Username: (자유)
  • Password: 앱 비밀번호
  • From email address: 보내는 사람 email 주소
  • To email address: 받는 사람 email 주소

와우! 테스트 메일이 잘 도착했습니다. 👏
